Petunjuk Penggunaan

Media Pembelajaran Interaktif Berbasis Web

Gunakan menu navigasi di bagian atas halaman untuk mengakses berbagai bagian website seperti Beranda,Petunjuk dan Menu.
Di Menu utama, Anda dapat melihat informasi umum tentang media pembelajaran interaktif yaitu, Tujuan, Materi, Video, Quiz, Identitas Pengembang
Klik menu "Tujuan" untuk mengetahui tujuan pembelajaran dan capaian pembelajaran. kemudia klik "kembali" jika ingin kembali ke menu utama
Klik menu "Materi" untuk menjelajahi berbagai topik pembelajaran. Pilih topik untuk melihat konten pembelajaran berupa teks, gambar. kemudia klik "kembali" jika ingin kembali ke menu utama
Klik Menu "video" untuk mempelajari mekanisme cara kerja sistem transmisi manual, kemudia klik "kembali" jika ingin kembali ke menu utama
Menu "Evaluasi" menyediakan soal-soal dan kuis interaktif untuk menguji pemahaman Anda terhadap materi yang dipelajari.
Klik Menu "Identitas" untuk mengetahui identitas pengembang dalam pembuatan website ini.kemudia klik "kembali" jika ingin kembali ke menu utama
